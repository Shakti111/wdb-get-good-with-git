    Typing Animations

    A good rule to follow is: if it can be done in CSS, then you shouldn't do it in JavaScript. JavaScript is far more taxing on the browser when it comes to animations than CSS. Even so, sometimes CSS just doesn't give the desired effect. I recently wanted to create a typing animation, and while the CSS solutions I found were somewhat effective, they weren't entirely convincing, and also fell apart on blocks of text spanning more than one line. For a character by character typing effect, I turned to JavaScript and it's yummy recursion. 

    let typeInterval;

const typeText = (text, element) => {
    if (text.length > 0) {
        element.innerText += text[0];

        typeInterval = setTimeout(function () {
            typeText(text.substring(1, text.length), element);
        }, 60);  
    }

    else return;
}

    The typeText function above takes two params, text, and an element. The breakout condition simply requires the length of text to be zero, in which case it's job is done. If the text length is greater than zero, the function will simply add whatever character sits at index zero within the string, and then uses setTimeout to call itself again, with the same element, and the same string, except this time missing the 0th character. I have used substring in this example but you can also use the slice method. I have set the timeout to 60ms as that's what I felt seemed convincing, although if you wanted to you could use a randomised number to make it even more realistic. Finally, the setTimeout is assigned to a variable outside of the functions scope, this is not enitrely necessary, but is handy if you want a way to cancel the animation part-way through, i.e. to trigger a different typing animation on the same element (don't forget that the setTimeout is just going to go off and do it's thing unless you call a clearTimeout to cancel it!).
